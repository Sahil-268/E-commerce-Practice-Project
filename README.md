# E-Commerce Practice Project

## Research:

### 1. Props (Properties)
> Used to pass data from the parent component to the child component.

### 2. State in React
> Used for the data that changes over time. It causes re-renders.

### 3. Props V/S States
| Props | States |
| :--- | :--- |
| It is passed from the parent | managed inside the component |
| Read only | Can Change |
| For communication | For dynamic data |

### 4. Lifecycle in React
> Different stages a component goes through is called the lifecycle of a component.
* **Mount:** The initial insertion of the component into the DOM.
* **Update:** When the component updates due to state or prop changes.
* **Unmount:** The removal of the component from the DOM.

### 5. UseEffect Hook
> Lets you perform side effects in the components.
* **API calls:** Fetching data from external endpoints.
* **Local storage:** Reading or writing client-side stored data.
* **Event listeners:** Attaching and detaching global event hooks.
* **Fetching data:** Updating component state with external resources.

### 6. Event Handling
> How React responds to the user's actions.
* **Button click:** Triggering actions on mouse clicks.
* **Hover:** Detecting cursor positioning over elements.
* **Typing:** Handling real-time text input fields.
* **Submit:** Processing form submissions safely.

### 7. Component Reusability
> Write once and use it many times with different data.

### 8. Infinite Scroll
> Automatically loads the data when the users reach the bottom of the page.

### 9. React Router
> Allows navigation between pages without refreshing the browser.

### 10. Responsive Design
> Making website look good on all screen sizes.
* **Mobile**
* **Tablet**
* **Laptops**
* **Desktops**
